# HOW DO I MAKE MY WEBSITE LIVE ON THE INTERNET?! 
You can use [Github Pages](https://docs.github.com/en/pages/quickstart)!

### crash course instructions 
1. set up your own, fresh GitHub repository. (to do this, you need to make a github account, then click the green "new repository" button
2. upload all your website files into your new repository (look for the link that says "upload an existing file")
3. go to Settings (at the top) -> Pages (on the left) -> look for the text that says,
"
### Branch
GitHub Pages is currently disabled. Select a source below to enable GitHub Pages for this repository."
4. Click the dropdown menu that currently says "None" and set it to "main" , then hit "Save"

This turns on GitHub pages! 
wait 5 minutes for Github to finish building your website
then the link to your new website should appear on this page. you can send it to your friends! (or post on edstem to share with the class! :D)
